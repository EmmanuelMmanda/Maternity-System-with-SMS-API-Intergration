*How to install system on your local Environment.
1. Copy maternity.zip into path ,C:\xampp\htdocs\
2.Extract zip file, (use winrar).=> New  path to be C:\xampp\htdocs\maternity
3.Open maternity.sql In %\Maternity\Database using Code editor 
4.XAMPP Phpmyadmin "localhost/phpmyadmin" and Create database named "maternity "
4.Copy all Sql Statements from maternity.sql on code editor
5.Open SQL section in PHPmyAdmin, Paste SQL Quries and Run 
6.Go to web Browser (Chrome pref.) and browse "localhost/maternity "
